## 题目描述
请实现有重复数字的有序数组的二分查找。
输出在数组中第一个大于等于查找值的位置，如果数组中不存在这样的数，则输出数组长度加一。
```python 
"""
在排序好的数组中使用二分查找
1) 定义左右边界类[l,r]
2) 判定 l<r
3) 找数,符合时返回,不符合移动左右边界
    注意:在重复数组中寻找第一个出现满足条件的数时:
        在移动的过程中需要特别注意边界值[+1,-1,+0]
        
"""
```
```python
def upper_find(n,v,a):
    l,r = 0,n
    while l<r:
        m = (l+r)//2
        if a[m] < v:
            l = m+1
        else:
            r = m
    return l+1
```
